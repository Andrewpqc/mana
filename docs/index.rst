如果你是 **flask** 的狂热粉，当你用 **flask构建大型项目** 时可能会觉得力不从心,
而mana的出现会改变这一切，使用 **flask with
mana**
开发项目可以让你方便的构建和管理flask项目，同时享受werkzeug和jinja的强大。

让我们开始 **flask with mana** 吧!
