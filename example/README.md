mana examples
===

## init_project

    $ mana init init_project

you can run by

    $ python manage.py runserver


## sqlinit_project

    $ mana sqlinit sqlinit_project
    $ python manage.py db init
    $ python manage.py db migrate -m "init"
    $ python manage.py db upgrade
    $ cd app
    $ mana blueprint auth

you can run by

    $ python manage.py runserver
