## What is mana
mana is a startapp command help flask developer fast start and manage their
project.

## Install mana
#### 1. install by pip

    $ pip install mana

#### 2. install from source

    $ git clone https://github.com/neo1218/mana
    $ cd mana
    $ sudo pip install .

## Mana command detail
1. mana init <project_name>
    * doc: <writing..>
2. mana sqlinit <project_name>
